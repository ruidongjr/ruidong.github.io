January 11, 2019

Deep Learning

We propose a deep learning (DL) architecture, where a digital twin of the real network environment is used to train the DL algorithm off-line at a central server. From the pre-trained deep neural network (DNN), the mobility management entity (MME) can obtain user association scheme in a real-time manner. Considering that real networks are not static, the MME further updates the DNN according to the variation of real networks.

October 1, 2018

Optimisation

We propose to maximize the network capacity by jointly optimizing the beamforming vectors and a binary vector that represents the user association decisions. A three-step Gaussian belief propagation (GaBP) based distributed solver is proposed to solve the non-convex NP-hard problem considered in this paper. By relaxing the binary optimisation variables, we first transform the optimization problem into a linear programming. Subsequently, GaBP is used to efficiently obtain a feasible solution in parallel, and two mapping algorithms are proposed to achieve different performance-complexity trade-offs.
